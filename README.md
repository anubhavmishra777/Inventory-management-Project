# Inventory-management-Project
Inventory Management System 📦
A lightweight and fast Inventory Management System built with Python, FastAPI, and SQLAlchemy.

✨ Key Features
Product Tracking: Add, Update, and Delete products.

Database: Managed via SQLAlchemy ORM.

Fast API: Automatic Swagger UI documentation.

🚀 Quick Start
Install requirements: pip install fastapi uvicorn sqlalchemy

Run server: uvicorn main:app --reload
